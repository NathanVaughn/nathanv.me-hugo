High-Performance Computing Cluster


I assisted in designing and constructing a compute cluster for PrISUm out of recycled hardware. The cluster consists of seven desktops with quad-core Intel Core i7 processors with 8GB of RAM, each running a build of CentOS. This gives the cluster a combined compute capacity of 28 CPU cores and 56GB of shared memory. All of the nodes are connected on a local gigabit ethernet network. The head node of the cluster runs Sun Grid Engine for job scheduling and has 1TB of network storage for applications and user workspace. The cluster is primarily used by the team for computational fluid dynamics simulations with STAR-CCM+ and MATLAB optimization problems. When the cluster is not in use, it contributes to community grid computing projects.

Date: Spring 2019
